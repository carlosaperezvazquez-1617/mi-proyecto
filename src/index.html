Actividad 1

El proyecto solicita lo siguiente:

- Crear (o usar) un repositorio en GitHub para un proyecto sencillo.

- Estructurar un proyecto básico con HTML y CSS.

- Realizar commits correctamente.

- Crear una rama adicional para agregar un pequeño cambio.

- Documentar el proyecto con un README sencillo.

Este proyecto utiliza GitHub como plataforma DevOps.
